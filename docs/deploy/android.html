안드로이드에 탑재하기¶

안드로이드 타겟을 위한 모델 빌드¶

안드로이드 타겟을 위한 모델의 Relay 컴파일은 android_rpc의 경우과 같은 절차를 따른다고 할 수 있습니다. 아래의 코드는 안드로이드 타겟이 필요로 하는 컴파일 결과물을 저장하게 됩니다.

lib.export_library("deploy_lib.so", ndk.create_shared)
with open("deploy_graph.json", "w") as fo:
    fo.write(graph.json())
with open("deploy_param.params", "wb") as fo:
    fo.write(relay.save_param_dict(params))

deploy_lib.so, deploy_graph.json, deploy_param.params 이 안드로이트 타겟에 올라가게 됩니다.

안드로이드 타겟을 위한 TVM 런타임¶

안드로이드 타겟 용으로 CPU/OpenCL 버전 친화적인 TVM 런타임을 빌드하려면 여기 를 참조하세요. 안드로이드 자바 TVM API로 모델을 읽어들이고 실행하려android java TVM API to load model & execute can be referred at this 자바 샘플 소스코드를 참조하세요.
