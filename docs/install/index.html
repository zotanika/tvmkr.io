설치¶

TVM을 설치하려면, 소스코드로 설치 을 읽어 보세요. 만일 모바일/임베디드 장치에 탑재하고자 한다면, 당신의 장치에 TVM 전체 스택을 설치할 필요는 없습니다. 단지 런타임만 설치하면 됩니다. 구현과 탑재 를 읽어 보세요. TVM을 빠르게 체험해 보거나 간단한 데모/튜토리얼을 원한다면, Docker Images 를 검토해 보세요.
