TVM 문서¶

TVM은 CPU, GPU 등의 범용 하드웨어 및 특수한 가속기들을 위한 공개 딥러닝 컴파일러 스택입니다. TVM의 목표는 생산성 위주의 딥러닝 프레임워크와 성능 혹은 효율성을 최우선 하는 하드웨어 백엔드 사이의 간극을 최대한 좁히는 것입니다.

시작하기¶

  • TVM을 설치하려면 instructions 을 따르세요.

  • Checkout the tutorials.

개발자 자료¶

  • 디자인과 아키텍쳐 는 TVM 아키텍쳐를 이해하고 이 프로젝트에 적극적으로 참여하고자 하는 개발자들에게 유용합니다.

  • Developer How-To Guide 에서 다양한 토픽에 대한 개발팁을 얻으세요.
